This game is a standard Checkers game with 2 human players.
The game follows standard Checkers rules, and the red player moves first.
All options are listed under the "File" menu. 
 - this includes 
    - new game
    - save game
    - load game
    - exit game
This game features a save/load game feature that allows the user to save the game and come back at a later time, resuming where they left it.
